# Проект: Giphy

### Описание проекта

- Командный проект для участия в соревнованиях.

**Используемые технологии**

- HTML
- CSS
- Flexbox
- Grid
- Responsive Design
- React
- Effector
- Vite

### Запуск проекта

Устанавливаем зависимости

```
npm install
```

Создать в корневой директории файл .env с ключом сайта https://developers.giphy.com/

```
VITE_GIPHY_KEY=apikeyhere
```

Запускаем проект

```
npm run dev
```
