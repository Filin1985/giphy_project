# Проект: Giphy

### Описание проекта

- Командный проект для участия в соревнованиях.

**Используемые технологии**

- HTML
- CSS
- Flexbox
- Grid
- Responsive Design
- Pixel Perfect
- React
- Vite

### Запуск проекта

Устанавливаем зависимости

```
npm install
```

Запускаем проект

```
npm run dev
```
